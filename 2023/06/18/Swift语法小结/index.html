循环

函数

​ 在参数名前 + “_” ：调用时省略参数名。在选择省不省略时要考虑 Function 名字是否足够清晰，在调用时能够明白用途。

​ 参数名前可 + 自定义词（例如，”of”）：增加代码可读性

嵌套函数

​ addFive 函数的作用域是 addTen 函数内，在此之外是无效的。 嵌套函数并不常用，但可以在编程过程中可以方便一些。

函数是一等公民

​ 可以把函数值定义成变量，之后就可以直接调用这个变量就可以进行操作。

闭包（匿名函数）

​ 闭包就是匿名函数

​ “$0”：匿名函数的第一个参数； “$1”：匿名函数的第二个参数，以此类推。”{}” 内的内容可以直接写成 { $0 < $1 }

​ **reduce(1, *)：**起始值是1，所有的运算都是 * ，从1开始，把每个数都乘一遍

​ “{}”中的内容是闭包，把它定义到了一个变量里，之后调用变量就可以做相应的操作。

可选值

​ optionalValue 要么是 nil 要么是一个值。

​ 使用情境：在进行网络请求时，request返回的东西不一定成功返回（网络断了，服务器挂了之类），返回的就可能是可选值，有值则正常运作，没有值则打印错误之类的。

​ “optionalValue!” 强制解包若 optionalValue 仍为 nil ，程序就会崩溃，App就挂掉啦！

​ “optionalValue ?? 0” 表示：若 optionalValue 有值则打印值；若 optionalValue 没有值，则打印 0。
